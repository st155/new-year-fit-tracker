Setup Steps

Step 1: Go to Terra Dashboard
Step 2: Navigate to Settings ‚Üí Webhooks
Step 3: Click Add Webhook Endpoint
Step 4: Enter the webhook URL from above
Step 5: Select events to subscribe to:
  • auth - User authentication events
  • activity - Workout and activity data
  • body - Body composition data
  • daily - Daily aggregate data
  • sleep - Sleep data
  • nutrition - Nutrition data
Step 6: Save the webhook configuration
Step 7: Test the webhook by:
  1. Going to Integrations page
  2. Clicking "Connect" on Terra integration
  3. Connecting any device (WHOOP, Garmin, etc.)
  4. Checking Edge Function logs for webhook events

Troubleshooting

No webhook events?
  • Verify the webhook URL is correctly configured in Terra Dashboard
  • Check that webhook is enabled and event types are selected
  • Ensure TERRA_SIGNING_SECRET is set in Supabase Edge Function secrets
  • Check Edge Function logs for any errors
  • Try disconnecting and reconnecting the device

Required Secrets

Make sure these secrets are configured in Supabase:

  • TERRA_API_KEY - Your Terra API key
  • TERRA_DEV_ID - Your Terra Developer ID
  • TERRA_SIGNING_SECRET - Your Terra webhook signing secret
